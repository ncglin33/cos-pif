# Build Project Workflow

This workflow describes the steps needed to prepare and build the project.
Since this is primarily a vanilla HTML/JS web project, the main requirement is to install any necessary dependencies.

## Steps

1. **Install Dependencies**
   Run the following command to install required Node.js packages:
   ```bash
   npm install
   ```

2. **Check Firebase Configuration**
   Ensure that the `firebase.json` and `.firebaserc` files are correctly set up.
