批次匯入毒理資料

上傳檔案 / 貼上文字 → 預覽與修正 → 最終寫入資料庫。

步驟一：下載範本、上傳檔案或貼上文字

方法一：下載標準 CSV 範本，依格式填寫後再上傳。支援 UTF-8 編碼，第一行必須是標頭，inciName 是唯一必要的欄位。

將 CSV 檔案拖曳至此，或點擊選擇

尚未選擇檔案

方法二：直接貼上 ChatGPT 產生的程式碼 / 文字

可貼上：
．已含標頭的 CSV 內容（建議直接貼 ```csv ... ``` 內文）
．只有一行一個 INCI 名稱的清單，本工具會自動轉為只有 inciName 欄的 CSV。

步驟二：預覽並修正

請仔細檢查資料。您可以直接點擊卡片內的欄位進行修改。系統將使用「標準化 ID」作為資料庫的唯一識別碼 (Document ID)。
