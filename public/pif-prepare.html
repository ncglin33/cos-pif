此頁僅作為資料集中與檢視完整性之用，不會自動送出 PIF。

一、產品與專案概況

先描述這個產品或系列的大方向，方便 PIF 製作者快速理解。

可簡述產品定位、系列名稱、主打特色，例如：敏感肌保濕精華、兒童防曬系列…
可同時列出中文與英文名稱，或多個品項清單。
例如：台灣一般通路、藥妝通路、電商、日本輸出等。
例如：一般成人、敏感肌、兒童 / 嬰幼兒等，搭配預定使用部位與頻率。

二、全成分與原料重點

可先貼上目前的全成分表（INCI + w/w% 或區間），以及關鍵原料與預定濃度，後續再細化為 PIF 各章。

建議依最終標籤順序列出全成分，可加上預定濃度或濃度區間，方便後續計算 SED 與 MOS。
例如：美白（熊果素、傳明酸）、防曬成分、防腐劑、酸類（AHA/BHA）等。
可列出已取得之 SDS、COA、TDS 或尚缺的文件，以便後續追蹤。
會依全成分欄位自動切出每一個成分產生陣列格式： [{ "order":1, "inciName":"Water (Aqua)", "percent":"qs", "note":"" }, …]
可直接複製到 pif03 匯入工具，或使用下方「下載 JSON 檔」。

三、宣稱與標示

先把行銷宣稱、標示需求與警語集中整理，方便後續比對法規。

例如：美白、淡斑、保濕、修護、舒緩、抗痘、防曬等，愈具體愈好。
例如：須準備繁中／英文標籤、日本版標示、INCI 清單呈現方式等。
例如：兒童需成人監護、避免接觸眼睛、噴霧類不可吸入等。

四、檢驗與文件現況

列出已完成與尚在規劃中的試驗、文件與製造工廠品質系統資訊，方便 PIF 製作者檢視缺口。

例如：安定性、微生物、PET、防曬 SPF / PA、保濕或美白功效等。
可作為與實驗室 / 檢驗單位溝通的待辦清單。
例如：特定原料 SDS、新版配方表、供應商聲明書、授權文件等。
記錄製造工廠名稱、地址、工廠登記字號、GMP / ISO 認證與適用範圍，以及委託或自設廠別等。

五、資料完整性檢視

系統依欄位是否有填寫，給出簡單的「完整度」提示，僅供人工檢視參考。

提示說明：● 綠色 = 已有基本內容； ● 橘色 = 有填但偏少； ● 紅色 = 尚未填寫。

可記錄與廠商、實驗室或內部討論重點，供 PIF 製作者參考。
